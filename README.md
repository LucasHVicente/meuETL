# meuETL
Projeto meu ETL do napp academy, o tema abordado são os casos de covid no brasil usando a api https://covid19-brazil-api-docs.vercel.app/

O projeto consiste em um script que busca os dados relacionados a casos de covid-19 nos estados brasileiros e gravar um arquivo csv.

Para rodar o script é necessário instalar as dependencias listadas no requirements.txt e em seguida executar o arquivo main.py com o comando:
```
 python main.py
```
